# لوحة تحكم متكاملة - Laravel 10

## نظرة عامة
لوحة تحكم احترافية مبنية باستخدام Laravel 10 مع نظام وحدات متكامل. تدعم اللوحة العديد من الميزات المتقدمة مثل نظام الصلاحيات، الترجمة المتعددة اللغات، رفع الملفات، والمزيد.

## المميزات الرئيسية

- **نظام وحدات متكامل** - إدارة سهلة للميزات كوحدات منفصلة
- **واجهة مستخدم تفاعلية** باستخدام Livewire 3 و Alpine.js
- **تصميم متجاوب** يعمل على جميع الأجهزة
- **نظام صلاحيات متقدم** باستخدام spatie/laravel-permission
- **دعم متعدد اللغات** مع spatie/laravel-translatable
- **إدارة الملفات** مع spatie/laravel-medialibrary
- **مصادقة متكاملة** مع Laravel Sanctum
- **واجهة برمجة التطبيقات (API)** مع توثيق Swagger
- **تحسين محركات البحث (SEO)**
- **لوحة معلومات تفاعلية** مع إحصائيات ورسوم بيانية

## المتطلبات التقنية

- PHP 8.1 أو أحدث
- Composer
- Node.js 16.x أو أحدث
- قاعدة بيانات (MySQL 5.7+ / PostgreSQL / SQLite / SQL Server)
- خادم ويب (Apache/Nginx)

## التثبيت

1. استنسخ المستودع:
   ```bash
   git clone [رابط المستودع]
   cd dashboard-master
   ```

2. قم بتثبيت حزم PHP:
   ```bash
   composer install
   ```

3. قم بإنشاء ملف `.env` من الملف النموذجي:
   ```bash
   cp .env.example .env
   ```

4. قم بإنشاء مفتاح التطبيق:
   ```bash
   php artisan key:generate
   ```

5. قم بتكوين إعدادات قاعدة البيانات في ملف `.env`:
   ```env
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=اسم_قاعدة_البيانات
   DB_USERNAME=اسم_المستخدم
   DB_PASSWORD=كلمة_المرور
   ```

6. قم بتشغيل الهجرات وبذر البيانات:
   ```bash
   php artisan migrate --seed
   ```

7. قم بربط مجلد التخزين:
   ```bash
   php artisan storage:link
   ```

8. قم بتثبيت حزم Node.js:
   ```bash
   npm install
   ```

9. قم ببناء الأصول:
   ```bash
   npm run dev
   # أو للبيئة الإنتاجية
   npm run prod
   ```

10. قم بتشغيل خادم التطوير:
    ```bash
    php artisan serve
    ```

## التخصيص

### إضافة لغة جديدة
1. أضف اللغة الجديدة إلى مجلد `resources/lang`
2. قم بتحديث ملف `config/app.php` لإضافة اللغة الجديدة
3. استخدم الدالة `__()` أو `trans()` لعرض النصوص المترجمة

### إضافة وحدة جديدة
```bash
php artisan module:make ModuleName
```
